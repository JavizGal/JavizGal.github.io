Sobre mí

Soy desarrollador web full stack con sólida experiencia en backend y en la creación de bases de datos eficientes. Me especializo en PHP, Laravel y MySQL, además de dominar HTML, CSS y JavaScript para el desarrollo de interfaces modernas, atractivas y funcionales.

Tengo experiencia en Java, Spring y Spring Boot, así como en el uso de PostgreSQL para bases de datos avanzadas. Manejo Git y GitHub para control de versiones y trabajo colaborativo, integrando buenas prácticas de desarrollo en entornos profesionales.

Cuento con conocimientos en Linux, lo que me permite administrar servidores y optimizar entornos de desarrollo. Además, he trabajado con C y ESP32, ampliando mis capacidades hacia soluciones que integran hardware y software. Mi enfoque está en crear proyectos de alta calidad, con buen rendimiento y adaptados a las necesidades reales de los usuarios.
